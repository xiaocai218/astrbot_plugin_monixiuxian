# astrbot_plugin_monixiuxian

修仙世界（AstrBot 插件），提供聊天指令 + Web 双端玩法，包含修炼、突破、签到、挂机、历练、装备、宗门、世界频道、排行榜、管理员面板等功能。

## 功能概览

| 模块 | 功能 |
| --- | --- |
| 聊天玩法 | 修炼、突破、签到、挂机/结算/取消、背包、使用道具、装备/卸下、回收、坊市交易、排行榜、在线列表 |
| 秘境历练 | Web 端五层秘境探索；安全通过、天灾、魔物、敌人、在线玩家遭遇战；支持功法耗灵、回合制战斗、献祭逃跑、1 血濒死警告、真实死亡与携宝重生 |
| 丹药系统 | 200 种丹药，5 品阶（凡/黄/玄/地/天）× 3 品级（下品/上品/无垢）；永久型（加血/攻击/防御/灵气/道韵/经验/血上限/突破率）和限时型（攻击/防御/灵气/修炼速度/全属性 buff）；下品/上品有副作用，无垢无副作用；天机阁按品阶概率出售，秘境掉落玄阶以下 |
| 成长系统 | 装备、心法、功法、灵气上限、道韵、高境界多小境界突破、功法修炼与遗忘 |
| 排行系统 | 修仙排行、死亡排行、在线列表 |
| Web 端 | 注册/登录、角色面板（含装备明细与丹药 buff 展示）、背包/坊市操作、世界频道、公告弹窗、秘境副本、PvP 遭遇战、死亡保留弹窗 |
| 宗门系统 | Web 端支持创建宗门、宗门列表分页浏览、加入/退出宗门、宗门简介编辑、成员列表与职位管理；已预留宗门仓库、炼器阁、炼丹阁、宗门商店、宗门任务入口 |
| 管理员 | 独立管理员账号密码登录；玩家资产编辑（含道韵）；武器/心法/功法/历练场景 CRUD；公告、坊市、IP 风控管理 |
| 渲染 | 使用 Pillow 生成帮助/面板/背包/排行等图片消息 |

| `enable_web` | `false` | 是否启用 Web 服务 |

## Web 使用说明

1. 开启 `enable_web` 后重启插件。
2. 访问 `http://<web_host>:<web_port>`。
3. 普通玩家使用访问密码进入后注册/登录角色。
4. 聊天端可通过 `/修仙 登录 <密钥>` 将 QQ 与 Web 角色绑定。
5. Web 端可体验秘境副本、在线玩家遭遇战、世界频道、宗门面板、公告弹窗、坊市与背包管理。
6. 世界频道会展示玩家所属宗门与宗门身份标识；历史消息持久化存储，并自动清理 30 天前的旧消息。
7. 若秘境中真实死亡，会弹出“携宝重生”选择框；需先完成选择后才能继续其它操作。
8. 管理员点击管理员登录按钮，使用 `web_admin_account` + `web_admin_password` 登录管理后台。
