# astrbot_plugin_xiuxian

修仙世界（AstrBot 插件），提供聊天指令 + Web 双端玩法，包含修炼、突破、签到、挂机、历练、装备、宗门、世界频道、排行榜、管理员面板等功能。

## 功能概览

| 模块 | 功能 |
| --- | --- |
| 修炼成长 | 修炼、突破（含小境界与道韵）、签到、挂机修炼、装备/心法/功法、灵气上限 |
| 秘境历练 | Web 端五层秘境探索，天灾/魔物/敌人/在线遭遇战，回合制战斗，真实死亡与携宝重生 |
| 丹药系统 | 200 种丹药（5 品阶 × 3 品级），永久属性型 + 限时 buff 型，副作用机制 |
| 宗门系统 | 创建/加入/退出/解散宗门，职位管理（宗主/副宗主/长老/弟子），宗门仓库（上交/兑换/贡献点/角色折扣/规则配置） |
| 交易系统 | 坊市上架/购买/下架，天机阁商店 |
| 社交系统 | 世界频道聊天（宗门徽标）、排行榜（修仙/死亡/在线） |
| 管理后台 | 玩家资产编辑，武器/心法/功法/场景/公告/坊市/IP 风控 CRUD |
| 渲染 | Pillow 图片消息（帮助/面板/背包/排行） |

## 主要配置

配置文件：`_conf_schema.json`

| 配置项 | 默认值 | 说明 |
| --- | --- | --- |
| `enable_web` | `false` | 是否启用 Web 服务 |
| `web_host` | `0.0.0.0` | 监听地址 |
| `web_port` | `8088` | 监听端口 |
| `web_access_password` | `""` | 访问密码，留空不校验 |
| `web_admin_account` | `admin` | 管理员账号 |
| `web_admin_password` | `""` | 管理员密码 |
| `cultivate_cooldown` | `60` | 修炼冷却（秒） |
| `afk_cultivate_max_minutes` | `60` | 挂机上限（分钟） |
| `adventure_cooldown` | `1800` | 历练冷却（秒） |

## 聊天指令

> 前缀：`/修仙`

| 指令 | 说明 |
| --- | --- |
| `帮助` | 显示所有可用指令 |
| `签到` | 每日签到领取奖励 |
| `面板` | 查看角色属性面板 |
| `修炼` | 修炼获取经验 |
| `挂机 <分钟>` | 挂机修炼 |
| `结算` / `取消挂机` | 领取/放弃挂机收益 |
| `突破` | 尝试突破当前境界 |
| `背包` | 查看背包物品 |
| `使用 <物品名>` | 使用消耗品 |
| `装备 <装备名>` / `卸下 <武器\|护甲>` | 装备/卸下 |
| `回收 <类型> <物品名> [数量]` | 回收物品获取灵石（类型：装备/物品/心法/功法） |
| `查看 <类型> <物品名>` | 查询物品详情（类型：装备/物品/心法/功法） |
| `坊市 [页码]` / `上架` / `购买` / `下架` | 坊市操作 |
| `排行` / `死亡排行` / `在线` | 排行榜 |
| `登录 <密钥>` / `登出` / `web` | Web 端绑定 |

## Web 使用

1. 开启 `enable_web` 后重启插件
2. 访问 `http://<host>:<port>`，注册/登录角色
3. 聊天端 `/修仙 登录 <密钥>` 绑定 QQ 角色
4. Web 端支持秘境副本、遭遇战、世界频道、宗门仓库、坊市、背包管理
5. 管理员使用独立账号密码登录管理后台
