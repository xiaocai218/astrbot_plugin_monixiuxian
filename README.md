# astrbot_plugin_xiuxian

修仙世界（AstrBot 插件），提供聊天指令 + Web 双端玩法，包含修炼、突破、签到、挂机、历练、装备、排行榜、管理员面板等功能。

## 版本信息

| 项目 | 内容 |
| --- | --- |
| 当前版本 | `v0.3.3` |
| 插件标识 | `astrbot_plugin_xiuxian` |
| 作者 | `xiuxian-dev` |
| 数据存储 | `SQLite`（`data/plugin_data/astrbot_plugin_xiuxian/xiuxian.db`） |

## 功能概览

| 模块 | 功能 |
| --- | --- |
| 聊天玩法 | 修炼、突破、签到、挂机/结算/取消、历练、背包、使用道具、装备/卸下 |
| 排行系统 | 修仙排行、死亡排行、在线列表 |
| Web 端 | 注册/登录、角色面板、背包操作、排行榜展示 |
| 管理员 | 独立管理员账号密码登录，玩家管理与运营操作（新增武器/心法/历练场景禁止重名） |
| 渲染 | 使用 Pillow 生成帮助/面板/背包/排行等图片消息 |

如你的 AstrBot 环境未自动安装插件依赖，请在插件目录执行：

```bash

```

## 主要配置（节选）

配置文件定义：`_conf_schema.json`

| 配置项 | 默认值 | 说明 |
| --- | --- | --- |
| `enable_web` | `false` | 是否启用 Web 服务 |
| `web_host` | `0.0.0.0` | Web 监听地址 |
| `web_port` | `8088` | Web 监听端口 |
| `web_access_password` | `""` | Web 访问密码；留空表示不校验 |
| `web_admin_account` | `admin` | 管理员登录账号 |
| `web_admin_password` | `""` | 管理员登录密码，建议单独设置 |
| `cultivate_cooldown` | `60` | 修炼冷却（秒） |
| `afk_cultivate_max_minutes` | `60` | 单次挂机上限（分钟） |
| `adventure_cooldown` | `1800` | 历练冷却（秒） |
| `checkin_*` | 多项 | 每日签到奖励范围与概率配置 |

## 聊天指令

> 指令前缀为：`/修仙`

| 指令 | 说明 |
| --- | --- |
| `/修仙 帮助` | 显示所有可用指令 |
| `/修仙 签到` | 每日签到领取奖励 |

| `/修仙 结算` | 领取挂机修炼收益 |
| `/修仙 取消挂机` | 取消挂机并放弃本次收益 |
| `/修仙 历练` | 外出历练（有冷却） |
| `/修仙 历练场景` | 查看历练场景列表 |

| `/修仙 使用 <物品名>` | 使用消耗品 |
| `/修仙 装备 <装备名>` | 装备武器或护甲 |

| `/修仙 web` | 获取网页版链接 |

## Web 使用说明

1. 开启 `enable_web` 后重启插件。
2. 访问 `http://<web_host>:<web_port>`。
3. 普通玩家使用访问密码进入后注册/登录角色。
4. 管理员点击管理员登录按钮，使用 `web_admin_account` + `web_admin_password` 登录。
5. 聊天端可通过 `/修仙 登录 <密钥>` 将 QQ 与 Web 角色绑定。
